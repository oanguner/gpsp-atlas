Installation
============

Requirements
------------

The GPSP helper modules require:

* NumPy
* SciPy
* Astropy
* Matplotlib

Install the required packages:

.. code-block:: bash

   pip install numpy scipy astropy matplotlib

Download
--------

Download the GPSP atlas FITS files and helper modules
from the project repository.

The atlas FITS files are distributed separately due to
their large size.

See the Downloads page for available atlas releases.
