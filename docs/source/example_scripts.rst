Example Scripts
===============

The following python scripts are provided for reproducing the examples shown throughout the GPSP documentation.

Galactic Bird's Eye View Map Generator
--------------------------------------

* Python script to generate bird's eye view survival probability maps of the Galaxy (:download:`galactic_birds_eye_view_map_generator.py <example_datasets/galactic_birds_eye_view_map_generator.py>`)

Galactic Plane Survival Probability Map Slice Generator
-------------------------------------------------------

* Python script to generate Galactic Plane survival probability map slice for a given energy and distance (:download:`gp_map_slice_generator.py <example_datasets/gp_map_slice_generator.py>`)
